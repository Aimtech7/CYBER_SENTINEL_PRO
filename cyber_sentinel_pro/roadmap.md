# Roadmap

## Short Term

- Expand Automation Engine settings UI for per-task JSON configuration
- Persist automation logs and export to CSV
- Add passive DNS/WHOIS, MISP, MalwareBazaar integrations
- MITRE ATT&CK mapping expansion across modules
- Add certificate/DNS health checks

## Medium Term

- RBAC and profile-gated settings
- Audit logging and safe-mode guardrails
- Offline AI summarizer and prompt library
- Notification integrations (email, Slack)

## Long Term

- Behavior profiling and anomaly detection models
- Workspace persistence and runbooks
- Command palette and onboarding overlays
